# **App Name**: CineStream

## Core Features:

- User Authentication: Secure user registration and login using Firebase Authentication.
- Video Streaming: Stream videos with access control based on user subscriptions or token balance.
- Content Categorization: Organize videos into categories and sub-categories for easy browsing.
- Subscription Management: Manage user subscriptions and token purchases via a payment gateway.
- Recommendation Engine: Suggest relevant videos to users based on their viewing history using a tool to extract topics from video metadata and other users' viewing history.
- Admin Dashboard: Admin interface to manage videos, categories, pricing, users, and view usage analytics.
- Data Storage: Store video metadata, user data, and watch history using Firestore.

## Style Guidelines:

- Primary color: Dark Indigo (#4F46E5) for a modern and premium feel.
- Background color: Dark Gray (#1F2937) to ensure the highlights pop.
- Accent color: Soft Lavender (#A78BFA) to complement and contrast the indigo, guiding the eye without harshness.
- Headline font: 'Poppins', a geometric sans-serif for headlines and shorter text.
- Body font: 'PT Sans', a humanist sans-serif for longer passages of text. Used to provide readability for extended content.
- Use 'lucide-react' icons for a consistent and clean visual language throughout the app.
- Responsive layout using Tailwind CSS to adapt to different screen sizes.
- Subtle transitions and animations to enhance user experience when navigating between pages.
